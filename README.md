# 绩点计算器

一个简单易用的绩点计算工具，帮助你快速计算平均绩点。

## 功能特点
- 输入课程名称、学分和绩点
- 实时计算平均绩点
- 美观友好的界面设计
- 支持结果分享和导出

## 使用方法
1. 访问在线版本：[https://用户名.github.io/gpa-calculator](https://用户名.github.io/gpa-calculator)
2. 输入你的姓名
3. 添加课程信息（名称、学分、绩点）
4. 点击"计算绩点"按钮获取结果

## 本地使用
如果你想在本地使用，只需下载`绩点计算器.html`文件并在浏览器中打开即可。
